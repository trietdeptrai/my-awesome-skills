# Adaptation paths

## Analyze the source

Create private working notes that identify:

- Repeated ideas, conflicts, images, and transformations.
- Claims or themes that can become decisions.
- Cause-and-effect relationships that can become mechanics.
- Misconceptions, tensions, or tradeoffs that can produce meaningful wrong choices.
- A chapter, section, or page anchor for every extracted element.

Rank candidates by importance to the book, suitability for player action, distinctness from other candidates, and relevance to the approved player outcome. Prefer 5–7 strong elements over exhaustive chapter coverage.

## Nonfiction path

Translate exposition into action:

| Source pattern | Useful player action |
|---|---|
| Principle or framework | Apply it to a new case |
| Causal explanation | Predict or manipulate an outcome |
| Comparison | Sort, match, or choose between strategies |
| Process | Sequence steps or recover from disruption |
| Tradeoff | Allocate limited resources and see consequences |
| Common misconception | Diagnose a plausible failure |

Use retrieval questions only as brief reinforcement. The main loop should require application. Make incorrect choices plausible and explain the relevant distinction in the resulting state.

## Fiction path

Extract a theme system rather than a plot summary:

- What values are in tension?
- What does the setting reward or punish?
- What relationships change under pressure?
- What information does each role possess?
- What kinds of decisions express the central theme?

When close adaptation rights are unconfirmed, design a new premise, cast, setting, chronology, scenes, and dialogue. Do not recreate famous sequences with renamed characters. Let the player's choices explore the selected themes through new consequences.

When the material is public domain or the user confirms ownership or adaptation rights, closer use of characters and plot can be proposed at Gate 1. Still write game-suitable dialogue and interactions instead of pasting prose.

## Source-fidelity test

For every quest, be able to answer:

1. Which source anchor supports this quest?
2. What must the player understand or experience?
3. Which action demonstrates that outcome?
4. How does feedback explain the consequence?
5. Does the quest introduce a claim the source does not support?

Mark deliberate interpretation as interpretation. Remove invented claims presented as the author's position.
